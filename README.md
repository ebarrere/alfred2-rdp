# Alfred 2 RDP Workflow

Quick-connect for iTap and MS RDP

## Usage:

* Type **rdp \<hostname/IP\>[:port]** (port defaults to 3389)
* Provide a "short hostname" (non-FQDN hostname) to auto-append the search suffix from `/etc/resolv.conf`
